# Word-Dict
英和辞典です


### 使い方
<br>
1.最新バージョンの**word-dict.deb**をダウンロード
<br>
2.sudo apt install ./**word-dict-{バージョン}.deb**でインストールする
<br>
3.**word-dict**で起動する
<br>
<br>
### 収録単語
<br>
収録されている単語はアルファベット順に**word_data.txt**に記載されています
<br>
<br>
