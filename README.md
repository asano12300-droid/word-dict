# Word-Dict
英和辞典です

## 可能なOS
ubuntu系linuxのみ

## 使い方
1.最新バージョンのダウンロード

```
wget https://github.com/asano12300-droid/word-dict/raw/refs/heads/main/(version)/word-dict-(version).deb
```
2.インストール
```
sudo apt install ./word-dict-(version).deb
```
3.起動する
```
word-dict
```

## 収録単語
収録単語:[詳細](./word_info.md)を確認!
