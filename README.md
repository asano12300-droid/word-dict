# Word-Dict
英和辞典です


## 使い方
1.最新バージョンのダウンロード
```
git clone https://github.com/asano12300-droid/word-dict/version/word-dict-version.deb
```
2.インストール
```
sudo apt install ./word-dict-version.deb
```
3.起動する
```
word-dict
```
